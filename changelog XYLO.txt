# XYLO KIT par Arko #
#################################


Le Xylo-Kit est un mini flavour mod ayant trait aux forêts.
La Forêt,lieu de mystères, terre privilégiée pour la chasse ou plus pragmatiquement source de revenus !

###v0.3 du 14 décembre 2012 pour CK2 v1.08

-révision du titre de Forestier : seule sa compétence en intendance compte pour son recrutement (retiré l'alternative de le nommer juste avec le trait Chasseur)

-vous pouvez couper du bois pour renflouer les caisses si vous êtes dans le rouge. Attention à ne pas trop tiré sur la corde quand même...
-ajout d'un event pour recruter votre Forestier, destiné à l'IA de base mais peu être utile si vous avez la flemme.
-ajout d'icônes pour toutes les spécialisations.
-ajout du titre "honorifique" : "Accusée de sorcellerie", à adresser à une courtisane. Sans conséquence pour le moment. 

###v0.2 du 02 décembre 2012 pour CK2 v1.08

La seconde voie impacte également les villes ; car les gouvernements de type "république" y auront accès également. En conséquence les bâtiment de la 2éme voie sont dispos dans les villes.
